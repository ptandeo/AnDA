# DESCRIPTION
Â¡AnDA! is a Python library for the Analog Data Assimilation. This fully data-driven approach aims at reconstructing the state of the system without knowing explicitly the dynamical model. Instead, a representative catalog of trajectories of the system is assumed to be available. AnDA combines analog forecasting methods with ensemble data assimilation.
